### Membres du groupe

###  Description
Application web permettant l’inscription, la connexion et la publication de messages en temps réel.

###  Fonctionnalités
- Authentification (email / mot de passe)
- Publication de messages
- Stockage Firestore (message, uid, email, timestamp)
- Affichage en temps réel

###  Technologies
- HTML / JavaScript  
- Firebase Authentication  
- Firebase Firestore
